# Hardware / DIY Electronics Дайджест 2026-07-21

## Ключові тези
- 2026 рік стає роком tool-changer 3D-принтерів: щонайменше 5 платформ оголошені або вийшли у перше півріччя

## Новини

- **Arduino VENTUNO Q — дебют на Embedded World 2026** — На виставці Embedded World 2026 представлено нову плату Arduino VENTUNO Q з процесором Qualcomm Dragonwing IQ8 і співпроцесором STM32H5. Демонстрація включала систему розпізнавання автомобілів на парковці у реальному часі. [Посилання](https://www.elektormagazine.com/news/embedded-world-2026-day-1)

- **Arduino UNO Q — перший Arduino з нативним Debian Linux (жовтень 2025)** — Плата поєднує Qualcomm Dragonwing QRB2210 для Linux-задач та STM32U585 для real-time керування. Це відкриває нові можливості для проектів на перетині вбудованих систем та ШІ. [Посилання](https://robocraze.com/blogs/post/arduino-vs-raspberry-pi-what-should-you-buy-in-2026)

- **PiZZa: Arduino-скетчі на Raspberry Pi Zero 2 W** — 1 липня 2026 Adafruit опублікував проект PiZZa, що дозволяє писати скетчі в Arduino IDE і запускати їх безпосередньо на Raspberry Pi Zero 2 W. Розмиває межу між двома платформами. [Посилання](https://blog.adafruit.com/2026/07/01/pizza-arduino-on-the-raspberry-pi-zero-2-w/)

Creality продовжує домінувати в сегменті Consumer 3D-друку. [Посилання](https://www.tomshardware.com/3d-printing/3d-printing-for-consumers-at-ces-2026-creality-snapmaker-atomform-and-protopasta-show-their-latest-systems)

[Посилання](https://www.tomshardware.com/3d-printing/3d-printing-for-consumers-at-ces-2026-creality-snapmaker-atomform-and-protopasta-show-their-latest-systems)
